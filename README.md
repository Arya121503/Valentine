# Valentine Website 💕

Website khusus untuk Valentine's Day dengan permintaan balikan yang romantis dan interaktif!

## Fitur 🌟

- Desain yang cantik dan romantis
- Animasi hati yang melayang
- Interaksi yang lucu (tombol "Tidak" akan lari!)
- Pesan-pesan yang menyentuh hati
- Halaman perayaan ketika jawaban "Ya!"

## Cara Menjalankan Lokal

1. Install dependencies:
```bash
npm install
```

2. Jalankan development server:
```bash
npm run dev
```

3. Buka [http://localhost:3000](http://localhost:3000) di browser

## Deploy ke GitHub Pages

Website ini sudah dikonfigurasi untuk deploy ke GitHub Pages menggunakan static export.

## Teknologi

- Next.js 14
- React 18
- CSS3 dengan animasi
